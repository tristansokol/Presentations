Item & Inventory Synchronization

Now I'll get into the Item syncing. You can see that WooCommerce isn't very happy because I haven't connected my Square Account. I'll do that now. This button will actually put me through the OAuth process. Here I need to log into my Square Account to approve the application, and then I'll get redirected back to my site. Now I'll need to select my Square location. I'm going to choose the succulent cart location and keep all of the defaults selected. Now that I have the location selected, I'll have the option to create a one time sync. Ideally you would have only one source of truth for all your items. Since I just created my WooCommerce account, I'm going to import my items from Square. Now WooCommerce is using Square's APIs to pull out the items from my account and creating identical products in WooCommerce. Now that the sync is completed. I'm going to navigate into my Products tab, and I see both of the products I had in my Square account, along with the all of the variations. Now, when I create a new item in WooCommerce, it is automatically created in my Square account as well. If I were to create a new item in my Square account, I would need to do another manual sync to get that item into my WooCommerce account. You can see here that I have some options in how my items get synced, including if my categories, inventory, and item images get synced between WooCommerce and Square.
