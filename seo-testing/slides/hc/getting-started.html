To get started with headless chrome, the first thing you'll notice is that you are going to start it from the command line instead of double clicking on something on your desktop. You probably don't want to dig into your computer to find your chrome executable, so the first thing you should think about is adding a handy alias. Then all you have to do is use the --headless command to get things started. --disable-gpu can fix some problems if you are on windows, in fact it might be required.
