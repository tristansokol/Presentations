Send a website as a PDF


					$ chrome --headless --print-to-pdf=report.pdf \
					--user-data-dir=./ https://www.google.com/

					$ uuencode report.pdf report.pdf | mail -s \
					"Just finished compiling the report" manager@compa.ny
Another pretty easy thing that comes to mind to automate is sending out reports. Have some analytics that you need to send your boss every Monday? Use a cron job to get headless chrome to save the website as a PDF, using the user data directory to save your authenticated cookies. Then use whatever program you'd like to mail that pdf, an take start taking Monday's off.
