What are tags, and how should I use them?
Tagging is one of the most basic essential peices of metadata for a question. It will fuel much of the data stuff I'll talk about in a bit, but the moral of the story is that anytime you talk about Stack Overflow, you should be talking about your tag. You want the developers to be tagging questions correctly so that you can find them, and luckily developers are incentivized to tag correctly since it helps gets answers faster. It is totally fine to create a tag for your company, and in fact, the reccommended approach would be to create a tag for each proeduct, and you can see that in for things like google maps, gmail, google cloud vision, each of those have their own tag and community around it. Personally, we are bit too small for something like that so we have one tag that we push users of all of our products to. How do you create a tag? It is easy, you just need enough points. It can also get tricky with managing open source products too.
